Closing and Self Closing

Closing Tags

HTML elements are written in start tag and end tags with content between

<tagname>content</tagname>

Most tags have opening and closing tags. Every tag must be closed properly.

  • <html>content</html>
  • <head>content</head>
  • <body>content</body>
  • <header>content</header>
  • <footer>content<footer>

Self-Closing Tags

Self-Closing tags are tags that are closed in opening tag.

Example

<input type="text" name=username value="User Name"/>

  • <area />
  • <base />
  • <br />
  • <col />
  • <command />
  • <embed />
  • <hr />
  • <img />
  • <input />
  • <keygen />
  • <link />
  • <meta />
  • <param />
  • <source />
  • <wbr />
