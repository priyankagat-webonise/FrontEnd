HTML char-set and meta tags

char-set

ASCII

It defined a unique binary 7-bits number for each storable character to support the numbers from 0-9, the upper/lower case English alphabet (a-z, A-Z), and some special characters like ! $ + - ( ) @ < > .It could only represent 128 different characters.

ANSI

It uses a full byte (8-bits) to represent 256 different characters.Since ANSI has been the default character set in Windows, it is supported by all browsers.

ISO-8859-1

ISO-8859-1 is an extension to ASCII, with added international characters. Like ANSI, it uses a full byte to represent twice as many characters than ASCII.

UTF-8

The Unicode Standard covers almost all the characters, punctuations, and symbols in the world.

Meta Tags

Meta elements are typically used to specify page description, keywords, author of the document, last modified, and other metadata.

charset Specifies the character encoding for the HTML document.
content Gives the value associated with the http-equiv or name attribute.
http-equiv Provides an HTTP header for the information/value of the content attribute.
name Specifies a name for the metadata
